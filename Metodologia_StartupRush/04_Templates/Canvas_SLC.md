# CANVAS SLC — STARTUP RUSH

> Preencha na Fase 2 (Sábado manhã). Este canvas define o que vocês vão construir
> e, igualmente importante, o que NÃO vão construir.
>
> **SLC = Simples, Adorável, Completo** ("Slick")
> - **Simples:** escopo limitado — construa apenas o necessário para testar a hipótese
> - **Adorável:** o que você construir deve ser algo que o cliente consiga interagir e apreciar
> - **Completo:** faz o que promete — v1 de algo simples, não v0.1 de algo quebrado

Antes de sair para testar, confirme:

- [ ] Hipótese está clara e escrita
