## 2. TIPO DE MVP ESCOLHIDO

- [ ] **MVP Fumaça (Landing Page)** — Página simples para medir interesse antes de construir
- [ ] **MVP Vídeo** — Demonstração visual da solução proposta
- [ ] **MVP Concierge** — Entregar o serviço manualmente para 1-3 clientes
- [ ] **MVP Mágico de Oz** — Interface que parece automática, mas vocês operam por trás
