## 5. ANÁLOGOS E ANTÍLOGOS

> Baseado no framework do investidor Randy Komisar (cap. 5 do livro, p.322-325).
> Ajuda a separar o que vocês **sabem** do que vocês **acham que sabem**.

### Análogos — "Isso já funcionou em outro lugar"

Que produto, empresa ou serviço existente prova que PARTE da nossa tese funciona? Não precisa ser o mesmo produto — basta provar algum pedaço.

| Análogo | O que prova? |
|---|---|
| Ex: Uber | Que pessoas confiam em estranhos via app para serviços |
| | |
| | |

### Antílogos — "Isso já falhou antes"

Que fracasso anterior mostra um risco para nossa tese? Empresas que tentaram algo parecido e deram errado — e por quê?

| Antílogo | Por que falhou? | O que faz o nosso caso ser diferente? |
|---|---|---|
| Ex: Juicero | Consumidores não pagam por tecnologia complexa para tarefa simples | |
| | | |
