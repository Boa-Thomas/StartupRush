## 1. OS 3 AS DAS MÉTRICAS (Checklist antes de usar qualquer número)

Para cada métrica que vocês coletaram, responda:

| Métrica que estamos usando | Acionável? | Acessível? | Auditável? |
|---|---|---|---|
| Ex: "12 de 30 pessoas disseram que pagariam R$50" | ✅ Sim | ✅ Sim | ✅ Sim |
| | [ ] Sim [ ] Não | [ ] Sim [ ] Não | [ ] Sim [ ] Não |
| | [ ] Sim [ ] Não | [ ] Sim [ ] Não | [ ] Sim [ ] Não |
| | [ ] Sim [ ] Não | [ ] Sim [ ] Não | [ ] Sim [ ] Não |

**Definições:**

- **Acionável:** A métrica mostra clara relação causa-efeito? Podemos agir com base nela? (Ex: "taxa de conversão subiu de 5% para 12%" → podemos agir. "500 likes" → não dá para agir)
- **Acessível:** Todo mundo na equipe entende o que essa métrica significa? Ou só o "analista de dados"?
- **Auditável:** Os dados vêm de interações reais com clientes reais? Dá para rastrear e verificar?

> **Regra:** Se uma métrica falha em QUALQUER um dos 3 As, ela é **métrica de vaidade**. Descarte ou transforme.

## 4. MÉTRICAS POR MOTOR DE CRESCIMENTO

Qual motor vocês estão apostando? (Ver [[Small Batches e Motores de Crescimento]])

- [ ] **Sticky (Retenção)** — A chave é medir se cliente volta/continua usando
- [ ] **Viral (Boca a boca)** — A chave é medir se cliente traz outros clientes
- [ ] **Paid (Marketing pago)** — A chave é medir se a receita cobre o custo de aquisição

### Se o motor é STICKY:

| Pergunta | Dado coletado |
|---|---|
| Quantas pessoas usaram o SLC? | _____ |
| Quantas usaram mais de 1 vez? | _____ |
| Taxa de retorno (%) | ___% |
| O que fez as pessoas voltarem? | ___________________________ |

**Critério de sucesso:** Taxa de retorno > 40% nas primeiras 24h = hipótese promissora.

### Se o motor é VIRAL:

| Pergunta | Dado coletado |
|---|---|
| Quantas pessoas testaram o SLC? | _____ |
| Quantas indicariam espontaneamente? | _____ |
| Quantas REALMENTE indicaram (trouxeram alguém)? | _____ |
| Coeficiente viral (indicações reais / pessoas) | _____ |

**Critério de sucesso:** Coeficiente viral > 1 = crescimento exponencial possível. Entre 0,5 e 1 = crescimento lento. Abaixo de 0,5 = não escala.

### Se o motor é PAID:

| Pergunta | Dado coletado |
|---|---|
| Quanto uma pessoa pagaria (LTV estimado)? | R$ _____ |
| Quanto custaria trazer essa pessoa (CAC)? | R$ _____ |
| LTV / CAC | _____ |

**Critério de sucesso:** LTV deve ser ≥ 3× CAC para o motor pago funcionar.

## 6. ARMADILHAS COMUNS (leia antes de concluir)

- 🚫 **"Mas aquela pessoa adorou!"** — Uma pessoa não é um padrão. Procure o que se REPETIU.
- 🚫 **"Vamos contar só as respostas positivas"** — Viés de confirmação. Conte TUDO.
- 🚫 **"Os dados não são claros, vamos confiar na intuição"** — Se os dados não são claros, a decisão correta é coletar mais dados, não chutar.
- 🚫 **"Gostei do resultado"** — Métricas não existem para fazer a gente se sentir bem. Existem para mostrar a verdade.
- 🚫 **"Mudamos tudo ao mesmo tempo"** — Se vocês mudaram 5 coisas entre coortes, não sabem qual mudança causou o efeito. Mude uma coisa por vez.
